## 🚀 ডেপ্লয়মেন্ট

### অপশন ১ — GitHub Pages (ফ্রি, সহজ)

```bash
# ১. রিপো ক্লোন করুন
git clone https://github.com/your-username/foodcallflow.git
cd foodcallflow

# ২. প্রোফাইল ইমেজ যোগ করুন
# profile-img.png ফাইলটি রুট ফোল্ডারে রাখুন

# ৩. GitHub-এ পুশ করুন
git add .
git commit -m "Initial deploy"
git push origin main
```

তারপর GitHub → Settings → Pages → Source: `main` branch → Save।

🌐 আপনার সাইট লাইভ হবে: `https://your-username.github.io/foodcallflow`

---

### অপশন ২ — Netlify (ড্র্যাগ & ড্রপ)

1. [netlify.com](https://netlify.com) → Log in
2. "Add new site" → "Deploy manually"
3. `index.html` এবং `profile-img.png` ফোল্ডারটি ড্র্যাগ করুন
4. ✅ লাইভ!
